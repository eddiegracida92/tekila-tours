# Ideas de proyectos para aprender Claude Code

## Idea 1: Generador de mensajes para redes sociales
Un script en Python que toma un tema corto (por ejemplo "evento de voluntarios el sábado")
y genera 3 versiones del mensaje: una formal, una casual y una con emojis.
**Lo aprenderías:** cómo crear archivos, correr scripts, y conectar Claude Code con código real.

## Idea 2: Página web personal o de tu organización
Una página HTML sencilla con tu nombre o el de tu organización, una descripción,
y links de contacto. Sin frameworks complicados — solo HTML y CSS limpio.
**Lo aprenderías:** cómo Claude Code crea archivos y estructura proyectos web desde cero.

## Idea 3: Rastreador de tareas en consola
Un pequeño programa que guarda tareas en un archivo de texto: puedes agregar una tarea,
marcarla como hecha, y ver la lista. Todo desde la terminal.
**Lo aprenderías:** cómo manejar archivos de datos y construir algo que realmente uses.
